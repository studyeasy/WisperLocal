# Packaging WisperLocal

Two artifacts can be produced:

1. **Standalone app** — `dist\WisperLocal\WisperLocal.exe` (no Python required).
2. **One-click installer** — `installer_output\WisperLocal-Setup-0.1.0.exe`
   (Start Menu entry, optional desktop/startup shortcuts, uninstaller).

## 1. Build the standalone app

```
scripts\build.bat
```

This runs PyInstaller against `WisperLocal.spec`, producing a self-contained
folder at `dist\WisperLocal\`. Double-click `WisperLocal.exe` to run — it needs
no Python install. Whisper models still download on first use and are cached
under `%USERPROFILE%\.cache\huggingface`.

## 2. Build the installer (optional)

Requires [Inno Setup 6](https://jrsoftware.org/isinfo.php):

```
winget install JRSoftware.InnoSetup
scripts\build-installer.bat
```

The resulting `WisperLocal-Setup-0.1.0.exe` is a per-user installer (no admin
required) that installs to `%LOCALAPPDATA%\Programs\WisperLocal`.

## Notes

- The build is CPU-only by default and large (~300–500 MB) because it bundles
  Qt and the CTranslate2/ONNX runtimes.
- To ship GPU support, install `nvidia-cublas-cu12` and `nvidia-cudnn-cu12` into
  the venv **before** building so their DLLs are collected.
- Icon is generated by `tools\gen_icon.py` into `assets\icon.ico`.
